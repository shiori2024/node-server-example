# node-server-example
A template project for a Node.js server.  
[Chinese](README.md) | [English](README-en.md)
## Description

A RESTful API server based on Node.js + Express + Sequelize + MySQL.

## Running the Project
```sh
# Install dependencies
npm install
# Start the server
node server.js
# or
nodemon .\server.js
```
