# node-server-example
一个Nodejs服务端模板项目。  
[中文说明](README.md) | [English](README-en.md)
## 项目简介

基于 Node.js + Express + Sequelize + MySQL 的RESTful API服务端。

## 项目运行
```sh
# 安装依赖
npm install
# 启动服务
node server.js
# or
nodemon .\server.js
```
